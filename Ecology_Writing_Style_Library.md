# Ecology Writing Style Library

> Purpose: A reference file kept inside the project. When polishing, the AI draws style cues ONLY from the "Preferred Samples" of the section that matches the current task (Methods task -> Methods samples).
> How it works: Two layers per section. **Preferred Samples** = 3-5 best passages the AI actually uses as style anchors. **Repository** = unlimited collection for your own browsing; the AI does NOT read these directly.
> Maintenance: Add passages whenever you read good writing. To make a passage "active", move it up into Preferred. Keep Preferred lean (3-5) so the style signal stays sharp.

---

## Selection Criteria (when choosing what to put in Preferred)

- From flagship journals (Ecology Letters, Ecology, Journal of Ecology, Methods in Ecology and Evolution, Functional Ecology, Global Ecology and Biogeography, etc.).
- Restrained sentences, conservative punctuation (few dashes), no inflated wording, appropriate hedging.
- Same function as the text you will polish (collect Methods passages to polish Methods).
- Within one Preferred set, keep the style consistent; avoid mixing very different voices.
